# MedSupply - Medical Equipment & Supplies Website

A modern, responsive front-end website for browsing and purchasing medical equipment and supplies.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Product Catalog**: 12 medical products across 4 categories
- **Search Functionality**: Real-time search across product names, descriptions, and categories
- **Category Filtering**: Filter products by Diagnostic, Protective Equipment, Monitoring, and Surgical categories
- **Interactive UI**: Smooth animations, hover effects, and dynamic cart management
- **Modern Design**: Clean, professional aesthetic with gradient backgrounds and card-based layouts

## Product Categories

- **Diagnostic Equipment**: Thermometers, stethoscopes, ECG machines
- **Protective Equipment**: N95 masks, surgical gloves, face shields
- **Monitoring Devices**: Blood pressure monitors, pulse oximeters, nebulizers
- **Surgical Instruments**: Surgical scissors, suture kits

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with flexbox, grid, and animations
- **JavaScript (Vanilla)**: Dynamic functionality without frameworks
- **Google Fonts**: Inter font family for clean typography

## How to Use

1. Open `index.html` in any modern web browser
2. Browse products in the catalog
3. Use the search bar to find specific items
4. Click category buttons to filter products
5. Click "Add to Cart" to add items (shows notification)
6. Click the cart icon to see cart count

## Features Breakdown

### Search
- Real-time filtering as you type
- Searches product names, descriptions, and categories
