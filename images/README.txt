Product Images Guide
====================

Place your product images in this folder with the following filenames:

1. blood-pressure-monitor.jpg    - Digital Blood Pressure Monitor
2. thermometer.jpg                - Infrared Thermometer
3. n95-mask.jpg                   - N95 Respirator Masks
4. pulse-oximeter.jpg             - Pulse Oximeter
5. stethoscope.jpg                - Digital Stethoscope
6. surgical-gloves.jpg            - Surgical Gloves
7. glucose-monitor.jpg            - Glucose Monitoring System
8. surgical-scissors.jpg          - Surgical Scissors Set
9. face-shields.jpg               - Medical Face Shields
10. ecg-machine.jpg               - ECG Machine
11. nebulizer.jpg                 - Nebulizer System
12. suture-kit.jpg                - Surgical Suture Kit

Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 800x600 pixels or similar aspect ratio
- File size: Keep under 500KB for fast loading
- Use clear, high-quality product photos

Note: If an image is missing, the website will show a placeholder icon instead.
You can also change the image filenames in script.js if you prefer different names.
